Warp Transform

In: Tools

Description

Warp your material using handles in the 2D view.

Parameters

Basic parameters

  • Grid:
    Adjust the number of divisions in the grid. Increasing the number of division will give you more control over the warp.
  • Show Grid: toggle
    When enabled, a grid will be displayed over the material to more easily view the effects of the filter
  • High Quality Sampling: toggle
    Enable to increase the sampling quality for better results.

Usage Guide

The Warp Transform filter lets you manually warp your material with handles in the 2D view. To use the Warp Transform filter:

  1. Add the Warp Transform filter to the top of the layer stack
  2. Adjust the number of handles with Basic parameters > Grid
  3. Drag handles to start warping the material

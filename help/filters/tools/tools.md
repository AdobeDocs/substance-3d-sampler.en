# Tools

Tools are the largest set of filters in Sampler, so it can be difficult to understand what sets them apart. The easiest way to understand tool filters is that they are used to fix technical issues with materials.

Many tools can be used on images and materials. It's generally a good idea to use filters to correct an image before converting it into part of the material. By correcting the image first, the tools only have to work on a single channel - the image data, while correcting the material requires correcting multiple channels. Only correcting a single channel decrease the amount of processing required, but also reduces the potential for errors.

Tool filters often make use of handles and other controls in the **2D view**, so it's often a good practice to have the **2D view** visible while working with tools.

Upscale

In: Tools

Description

The Upscale filter uses AI to upsample the PBR channels (BaseColor, Roughness, Normal, Metallic, Height) from the layers below it.

In this exemple we start with a 1024x1024px image but the output result is in 4098x4098px. The results using the Upscale filter are more defined.

Note

Advanced filter

The Upscale is an advance filter.
To use it at it's maximum capacity and avoiding blurry results, we recommend to set the layers below the Upscale in Layer Input Max or Layer Input Min. 

There are no limits on how many Upscale filters can be used, but upsampling above 8k resolution might significantly impact performance.

Parameters

Basic parameters

  • Up sample: Toggle button group
    Choose the multiplication factor to upscale

How to

In the image above, a low resolution image is processed by the Image to Material (AI Powered).

The Upscale filter is added to up sample the results. It halucinate details in order to reach a higher resolution keeping the quality of the material. You can chose in the properties to upsample by 2 or by 4.

## Parameters

<b>Basic parameters</b>

* <b>Input Tiled</b>: Toggle  
  Enable this setting to ensure this filter doesn't break tiling materials.
* <b>Radius</b>: 0-50  
  Adjust the radius to control the size of the folds that the filter tries to minimize. A low radius will smooth out finer details.
* <b>Local details</b>: 0-1  
  Use a higher value to maintain local details while still removing folds.

<b>Normal</b>

* <b>Override common parameters</b>: Toggle  
  Control the parameters for this channel individually.

<b>Height</b>

* <b>Override common parameters</b>: Toggle  
  Control the parameters for this channel individually.

<b>Mask</b>

* <b>Custom mask</b>: Toggle  
  Enable use of a custom mask to control where the effects of this filter appear.
* <b>Custom mask</b>: Brush/image select  
  Create or select a custom mask.
* <b>Custom mask invert</b>: Toggle  
  Invert the mask.

Panorama Patch

In: HDRI Tools

Description

Patch problem areas in your environment light. Panorama patch maintains the transformation and skewing to match the patch with the background surface naturally.

The images below show the Panorama Patch in action - in this case, replacing the center of a large cloud with open skies.

Parameters

Basic parameters

  • Patch Radius: 0-1
    Adjust the size of the patch.
  • Patch Contrast: 0-1
    Adjust the hardness of the edges of the patch.
  • Patch Rotation: 0-1
    Rotate the patched area.

Usage Guide

Add the Panorama Patch filter to the top of the layer stack, and then use the handles in the 2D view to adjust the source and patch locations.

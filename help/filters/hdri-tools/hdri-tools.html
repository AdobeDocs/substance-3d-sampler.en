HDRI Tools

Future change

Support for environment lights and meshes will be removed with the release of Sampler V5.2. The removal of environment lights will also include the removal of HDRI tools and filters.

Filters in the HDRI Tools section are tools specifically for working with environment lights. HDRI stands for High Dynamic Range Image, but is commonly used to refer to the panoramic images used to create environment lights. Modifying panoramic images in traditional software can be difficult due to how spherical 3D information must be mapped to a 2D image file - the HDRI Tools in Sampler are designed with this consideration in mind, so patching and adding lights to your HDRI's is much easier.

The following filters are HDRI Tools:

  • Color Temperature Adjustment
  • Exposure
  • Exposure Preview
  • HDR Merge
  • Line Light
  • Nadir Extract
  • Nadir Patch
  • Panorama Patch
  • Plane Light
  • Shape Light
  • Sphere Light
  • Straighten Horizon

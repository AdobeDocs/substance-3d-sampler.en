Send to any app

From the 4.5 Sampler release you can now send your assets from Sampler directly to any app where the Connector has been implememented.
It allows to send assets from Sampler to a third party applications in one click, to avoid having to go through the manual export and import process and gaining time.

 

Today the Send-to is available in Blender, Unreal Engine, Unity, 3ds Max, and Maya though the Substance Plugins. 

Supported app versions

To be able to use the Send-to it is necessary to have the right version fo Sampler, of the third-party application and of the plugins.

Here are the minimum versions you need to be able to use the Send-to:

  • Unreal Engine Plugin 
    • Anything after UE5.4 is fully supported.
    • Released on the UE marketplace
  • Maya Plugin v3.0.0+
    • Supported in Maya 2025, 2024, 2023, and 2022
    • Available out of the box since Maya 2025, for previous versions the plugin needs to be downloaded from the website
  • 3ds Max Plugin v3.0.0+
    • Supported in 3ds Max 2025, 2024, 2023, and 2022
    • Available out of the box since 3ds Max 2025,  for previous versions the plugin needs to be downloaded from the website
  • Unity Plugin v3.11+
    • Unity versions 2022, 2021 or 2020
    • The plugin needs to be downloaded from the Asset Store
  • Blender Plugin v2.1+
    • Blender versions 3.0 and above
    • The plugin is available to be downloaded on the website

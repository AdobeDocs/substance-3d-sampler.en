# Crash when using the Image to Material or Delighter

The **Image To Material (AI Powered)** and **Delighter** filters require a lot of available VRAM (at least 1GB).

If you have a GPU Card with only 2GB of VRAM available and are working in 2K/4K resolution, we won't be able to allocate enough memory to execute the filter and a crash may occurs.

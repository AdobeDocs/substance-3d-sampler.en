# Import Substance Alchemist projects in Substance 3D Sampler

Starting with version 3.0.0, the previous Substance Alchemist projects are not automatically available. This pages describes how to retrieve them.

## 1 - Launch Substance Alchemist

1. Open the project of your choice in Alchemist
1. Click on **File** &gt; **Export project...**
1. Choose the project location and click **OK**

After that action a **.alch** file with a **dependencies** folder will be available.

### 2 - Launch Substance 3D Sampler

1. On the **Welcome Screen**, click **Open**
1. Go to the folder where the **.alch** file is located.
1. Open the **.alch** file (be sure the **dependencies** folder is still next to the .alch file).

All your materials will be listed in the Project panel under the Materials section.

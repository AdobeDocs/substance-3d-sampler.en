Project panel

Projects in Sampler act like packages that can store multiple assets. The Project panel displays the assets that make up your current project.

Controls at the top of the Project panel allow you to add or manage assets in your current project:

  • Use the Add button to create a new Material or Environment light.
  • Use the Duplicate button to duplicate the currently selected asset.
  • Use the Rename button to rename the currently selected asset.
  • Use the Delete button to delete the currently selected asset.
Note

You can also right click any asset to Rename, Delete, or Duplicate it; or open the assets file location.

Click one of your assets to load them in the Viewport, or drag an asset onto the Layers panel to use it in other compatible assets.

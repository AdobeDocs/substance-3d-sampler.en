# Assets panel

![](../../../assets/Assets-panel-gen.png.img.png)

The <b>Assets panel</b> holds assets that you can use to build your creations. Sampler includes a collection of materials, filters, and texture generators to help you get started.

The <b>Assets panel </b>has some controls to help organize and find assets:

* Use the search bar to find assets quickly.
* Filter assets by type.
* Group assets by type or by category.
* Switch between list and icon view.

## Add assets to the Assets panel

To add your own assets to the assets panel, click the <b>+ </b>at the lower right of the <b>Assets panel</b>. Browse to the location of your asset, select it, and click <b>Open</b>. You can remove custom assets from the <b>Assets panel</b> by dragging them over the trash can.

>[!NOTE]
>
> Only custom assets can be deleted in the<b> Assets panel. </b>It is not possible to delete Sampler's default assets.

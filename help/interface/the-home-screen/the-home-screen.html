The Home Screen

When you open Sampler, the Home Screen will appear. The Home Screen has a number of options to help you get started with a new or existing Sampler project.

  1. New project: Create a new project by importing a file and choosing from a collection of Quick actions.
  2. Empty project: Create a new empty project.
  3. Open: Open a project with your system's file browser.
  4. Home: Access recommended tutorials, create a new project, or view a list of recent projects.
  5. Learn: Access Sampler's video tutorials and learning content.
  6. Files: View a list of recently opened projects.
  7. About Sampler: Find out more about your version of 3D Sampler.
  8. Documentation: View the documentation and expand your abilities.

When you open or start a new project, the Home Screen will disappear so you can start creating assets. You can open the Home Screen again from the File menu.

# Fonts are not displayed correctly

If you're on Windows 10 Enterprise and use the [Block Untrusted Fonts feature](https://docs.microsoft.com/en-us/windows/security/threat-protection/block-untrusted-fonts-in-enterprise), you can follow this [tutorial](https://www.tenforums.com/tutorials/139148-exclude-specific-apps-untrusted-font-blocking-windows-10-a.html) to exclude Sampler from the list of untrusted apps for font blocking.

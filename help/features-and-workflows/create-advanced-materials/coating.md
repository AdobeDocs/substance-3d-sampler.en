# Work with the coating channel

As the name suggests, the coating channel lets you add a coating to your material. Coatings can have their own roughness value, which can affect the base roughness of the material, helping to create effects like a varnished or polished surface. Watch the video below to learn how to use the coating channel in Sampler to create a varnished parquet floor material from scratch.

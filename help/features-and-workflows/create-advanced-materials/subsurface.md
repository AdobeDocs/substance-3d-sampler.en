# Work with subsurface channels

Subsurface channels are useful for opaque but translucent materials, mimicking the effect of light scattering below the surface of the material. This effect commonly happens with materials like skin, leaves, and many plastics. Watch the video below to learn more about the subsurface parameters and use them to create a resin material.

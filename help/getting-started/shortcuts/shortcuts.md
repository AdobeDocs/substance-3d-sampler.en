# Shortcuts

This page list all the keyboard shortcut available and their actions.

## Global Shortcuts

| *Action* | *Windows* | *MacOS* |
| --- | --- | --- |
| Undo | Ctrl + Z | ⌘ + Z |
| Redo | Ctrl + Shift + Z | ⌘ + Shift + Z |
| Save | Ctrl + S | ⌘ + S |
| Save As | Ctrl + Shift + S | ⌘ + Shift + S |
| Export | Ctrl + E | ⌘ + E |
| 3D View only | 3 | 3 |
| 2D View only | 2 | 2 |

## 3D View Shortcuts

| *Action* | *Windows* | *MacOS* |
| --- | --- | --- |
| Camera Rotation | Alt + Mouse Left  or Mouse Left | Option + Mouse Left  or Mouse Left |
| Camera Pan | Alt + Mouse Middle  or Mouse Middle | Option + Mouse Middle  or Mouse Middle |
| Camera Zoom | Alt + Mouse Right  or Mouse Wheel | Option + Mouse Right  or Mouse Wheel |
| Focus | F | F |
| Environment Rotation | Shift + Mouse Right | Shift + Mouse Right |
| Environment Visibility | 1 | 1 |

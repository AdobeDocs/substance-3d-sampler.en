Import resources

Sampler can use external resources like images and Substance files to modify your project. Use any of the following options to import a file to your project:

  • Drag and drop a file from your file explorer onto the Sampler window. An import window appears with options to change how the import is handled.

  • From the Left bar use the Get content button, then select either Import in Layer Stack, or Import in Your Assets. Both options will open a file explorer where you can navigate to and select the file or files to import.
    • Import in Layer Stack imports the file for your current project.
    • Import in Your Assets imports the file so it can be accessed from any project.

  • In the Layers panel, if no layers have been created, you can use the available links to import a file to form the base of your material.
Note

Resources are linked and not imported. As a consequence, if a resource file is moved or deleted, materials or content created with that resource will be affected. As a result, we recommend using a dedicated folder for Sampler Assets.

By default, Sampler's included assets are stored under C:\Program Files\Adobe\Adobe Substance 3D Sampler\Resources\assets

Supported file formats

Type Description

Bitmaps / Images
(JPEG, PNG, etc)

 You can drag and drop or import your bitmaps in the Layers panel, or into SBSAR or filter parameters that allow an image input.
Substance Packages
(SBSAR)		 You can import SBSAR files into the Assets panel or into the Layers panel.
